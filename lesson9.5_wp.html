1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. Для работы с препроцессорами css и другими технологиями в webpack существуют Loaders(это отдельные пакеты которые надо установить)
    офф сайт https://webpack.js.org/loaders/

  3. Создадим в папке src папку css в ней index.css

  4. Установим css-loader команда: npm install --save-dev css-loader
    Loaders не нужно импортировать, а нужно сразу описывать в webpack.config.js в module.exports (описание свойств https://webpack.js.org/loaders/css-loader/ )

  5. Далее в главном файле js index.js нужно подключить css импортом import './css/index.css'

  6. Но после этого стили не применились т.к. css-loader только загружает стили css,
    а что бы приметить стили нужно установить пакет style-loader команда npm install --save-dev style-loader и в webpack.config.js в module.exports в module: use: ['style-loader', 'css-loader']

  7. Создадим в папке css файл header.css и импортируем его в файл index.css так подключаются другие css файлы в главный css файл

  8. Но все наши стили при этом вставляются в тег head в html и это плохо выглядит.
