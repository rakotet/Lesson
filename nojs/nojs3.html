1. NPM это программа node.js которая позволяет работать с пакетами и модулями node.js

  2. Для создания json файла с зависимостями (файл с инструкциями) нашего проэкта в папке с проэктом в консоле пишем npm init и заполняем

  3. Делее нам необходимо установить web server express https://www.npmjs.com/package/express
    Там указаны команды как установить данный модуль через npm (npm install express --save)
    флаг --save означает что мы хотим установить данный пакет как зависимость к нашему проэкту. В папке проэкта создастся папка node_modules в которой будут установленные пакеты.

  4. Далее нужно установить пакет nodemon - нужен для удобной разработки, будет постоянно перезагружать сервер.
    Команда npm i nodemon --save-dev (-dev зависимость для разработки, не включается в функционал приложения)

  5. Так же требуется установить эти пакеты глобально. Команда npm i nodemon -g (-g флаг означает глобально)

  6. Если мы удалил папку node_modules или какой то пакет в ней, то всё можно востановить через наш файл с зависимостями package.json Для этого в консоле папки команда: npm i
    И все наши зависимости будут установленны обратно
    Так же эта команда нужна для переноса проэкта, что бы установить все зависимости на новом месте

  7. package-lock.json это системный файл ускоряющий работу npm и его трогать не надо!!!!
