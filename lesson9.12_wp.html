1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. Создаём в папке modules файл ts.module.ts и напишем в нем простой код используя TypeScript

  3. Устанавливаем пакет npm install --save-dev @babel/preset-typescript
    И настраиваем по оф документации https://babeljs.io/docs/en/babel-preset-typescript#docsNav

  4. Готово! Можно собирать проэкт, команда: npm run build
