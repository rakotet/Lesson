1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. Для подключения стилей css в отдельном файле используется npm mini-css-extract-plugin, команда: npm install --save-dev mini-css-extract-plugin

  3. Подключаем в webpack.config.js в const MiniCssExtractPlugin = require('mini-css-extract-plugin');

  4. И прописуем его свойства по офф документации в файл webpack.config.js.
    Теперь при сборке проэкта все наши css файлы будут объеденяться в один dist/style.css и он будет подключен к проэкту
    Но при этом файл style.css не оптимизирован (содержит лишние пробелы и т.д.)

  5. Для оптимизации css файлов нужно установить плагин: npm install --save-dev optimize-css-assets-webpack-plugin
    И настроить его свойства по офф документации. Так же необходимо установить плагин: npm install terser-webpack-plugin --save-dev
    И настроить его свойства по офф документации.

  6. Но есть одна загвоздка, uglifyjs-webpack-plugin работает только с js5 синтаксисом, и для конвертации нашего кода в js5 код нужно настроить Babel (чуть позже настроим)

  7. Теперь при итоговой сборке проэкта (npm run build) css и html файлы будут оптимизированны
