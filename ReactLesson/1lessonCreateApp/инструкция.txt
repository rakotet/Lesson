1) Что бы создать React проэкт, сначала нужно установить глобально npm i -g create-react-app (перед этим установить node.js и npm)
2) Открыть папку для проэкта в констоли и ввести npx create-react-app "название проэкта"
3) Для управления псевдоэлементами и классами CSS используется библиотека Radium, установка npm install radium
4) Создаём отдельный компанент ErrorBoundary для того что бы он отлавливал некоторые ошибки
5) Библиотека prop-types обеспечивает валидацию параметров в React, что бы установить npm i prop-types
6) В консоле в папке проэкта npm run eject затем npm i это открывает возможности настройки стандартного webpack встоенного в React
7) Для роутинга в реакте используется пакет react-router-dom, устанавливаем его
8) Что бы роутинг заработал нужно обернуть наше приложение в компанент из библиотеки роутинга BrowserRouter (в index.js), и на каждую ссылку навешивать Route
9) Что бы страница не перезагружалась при переходе по ссылкам, нужно тег <a/> заменить на компанент NavLink
