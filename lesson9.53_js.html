Все файлы приложения находятся в папке js-application


  1. Устанавливаем WebPack, WebPack-cli, WebPack-dev-server как зависимости для разработки

  2. Далее создаём файл конфигурации webpack.config.js и заполняем его согласно нашим зависимостям

  3. Далее устанавливаем npm i --save-dev html-webpack-plugin@next и настраиваем соглассно офф документации

  4. Пропишем некоторые скрипты в package.json для ускорения набирания команд webpack

  5. Далее установим и настроим npm install --save-dev @babel/core @babel/cli @babel/preset-env
    npm install --save @babel/polyfill
    npm install -D babel-loader

  6. Далее в корне проэкта создадим файл .babelrc и заполним его соглассно офф документации
