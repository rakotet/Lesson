1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. Существует очень популярная FrontEnd библиотека для JavaScript она называется React.js и она использует особый синтаксис JSX,
    JSX позволяет писать различные HTML теги прямо в коде JS, но нативный js этого не понимает,
    Но webpack позваляет преобразовать JSX в обычный js используя различные loaders

  3. Сначала установим React команда npm i -S react react-dom

  4. Подключаем react в index.js, import React from 'react' import {render} from 'react-dom'

  5. Создадим файл App.js и напишем в нем класический React компонент

  6. Что бы компилировать в webpack файлы React необходимо установить npm install --save-dev @babel/preset-react
    И настроить его по офф документации https://babeljs.io/docs/en/babel-preset-react#docsNav

  7. Удаляем header.component.js и весь его код переносим в App.js с использованием синтаксиса JSX
