WebPack это программа которая переводит все препроцессоры в нативный css (т.к. браузеры понимают только нативный css) и систиматизирует файлы js и папки с файлами

Для настройки и установки WebPack надо:

  1. Установить node.js с оф сайта https://nodejs.org/en/
  2. Проверить установился ли node.js: в cmd или терминале команда node -v и npm -v должно написать установленную версию
  3. Через npm инициализируем пакет json в нашем проэкте: в cmd или терминале папки проэкта команда npm init
  4. Далее в консоле (терминале) пишем имя паета, версию(можно оставить), описание, начальный скрипт, тест команда(можно оставить), гит репозиторий,
    ключевые слова(как в описании к html), автор, лицензия(можно оставить), enter
  5. Появляется в папке package.json в котором будем описывать зависимости для нашего проэкта и писать некоторые скрипты
    для автоматизации некотрых процессов
  6. Устанавливаем сам WebPack с офф сайта https://webpack.js.org/
  7. На сайте DOCUMENTATION -> GUIDES -> Installation -> в cmd или терминале в папке проэкта npm install --save-dev webpack webpack-cli
    webpack работает на платформе node.js и не включает себя в финальную сборку проэкта
