Основные команды Git


  1. Инициировать локальные репозиторий: git init
  2. Добавить измененные файлы: git add .
  3. Убрать индексированные файлы: git reset
  4. Файлы ждущие коммит: git status
  5. Закоментить и сохранить измененные файлы в локальном репозитории: git commit -m "Описание изменений"
  6. Сделать откат к предыдущей версии: git reset --hard idCommita
  7. Посмотреть историю изменений и коммиты: git log
  8. Краткий лог коммитов: git log --oneline
  9. Подключить удаленный репозиторий: git remote add origin https://github.com/libgit2/libgit2
  10. Посмотреть какой удаленный репозиторий подключен: git remote
  11. Клонирование существующего репозитория: git clone https://github.com/libgit2/libgit2
  12. Загрузить данные из удаленного репозитория: git pull
  13. Загрузить данные на удаленный репозиторий: git push -u origin master
  14. Посмотреть текущию ветку: git branch
  15. Создание новой ветки: git branch testing
  16. Переключение веток: git checkout testing
  17. Посмотреть все ветки: git branch -a
  18. Объединение текущей ветки с другой: git merge testing
