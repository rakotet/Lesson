1. Плагины для wevpack это отдельно подключаемые классы которые добавляют дополнительный функционал

  2. Описание плагинов есть на офф сайте wevpack https://webpack.js.org/plugins/

  3. HtmlWebpackPlugin - служит для создание HTML страниц и подключения к ним скриптов

  4. Установка плагина npm install --save-dev html-webpack-plugin

  5. Подключаем плагин в webpack.config.js в plugins

  6. Теперь при сборке проэкта webpack будет использовать наши html шаблоны
