1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. Babel - это инструмент который переводит весь наш код приложения в ES5 синтаксис (что бы работало во всех браузерах и все старые плагины)

  3. Устанавливаем Babel: npm install --save-dev @babel/core @babel/cli @babel/preset-env
    npm install --save @babel/polyfill
    npm install -D babel-loader

  4. Далее необходимо задать свойства расписанные на офф сайте https://babeljs.io/setup#installation
    В файле webpack.config.js

  5. Далее создаём файл в корне проэкта .babelrc это конфигурационный файл самого babel и заполняем его
