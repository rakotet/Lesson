1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. webpack-dev-server служит для сборки проэкта "на лету" при изменении файлов проэкта (более удобная разработка)

  3. Установим npm install -D webpack-dev-server
    И в webpack.config.js в корне module.exports пропишем свойства devServer

  4. В package.json пропишем новый скрипт start для запуска dev-server в режиме разработки

  5. Для запуска dev-server команда: npm run start
