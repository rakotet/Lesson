1. Смотреть настроенный webpack с комментариями в папке webpack-lesson

  2. В файле package.json есть "scripts" в которых описываются нами написанные команды управления webpack

  3. build - это команда сборки проэкта с последующей оптимизацией его, dev - вход в режим разработки и просмотра исходного кода в файле bundle.js

  4. В папке modules лежат 3 файла с различными возможными импортами и экспортами модулей в js и webpack. Экспортируем модули из этих файлов, а импортируем в файл index.js лежащий в папке src

  5. Что бы проверить что всё работает в консоле node dist/bundle.js запускаем наш файл который сделал сборщик webpack
